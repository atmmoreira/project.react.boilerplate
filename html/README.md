# Boilerplate HTML, CSS and Javascript
This boilerplate it's for learning technologies like: Python, Django, Flask, Node (Nest.js), React (Next.js) and Vue (Nuxt.js).

<br />

# Technologies
 - HTML, CSS, TailwindCSS and Javascript.
 - DevDependencies:
    - @babel/cli
    - @babel/core
    - @babel/preset-env
    - babel-loader
    - css-loader
    - postcss
    - postcss-loader
    - postcss-preset-env
    - style-loader
    - tailwindcss
    - webpack
    - webpack-cli

You can use this command for install devDependencies

    npm install @babel/cli @babel/core @babel/preset-env babel-loader css-loader postcss postcss-loader postcss-preset-env style-loader tailwindcss webpack webpack-cli -D
